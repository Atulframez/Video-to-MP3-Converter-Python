🎵 Video to MP3 Converter (Python)

A lightweight and efficient Video to MP3 Converter built with Python.
This application allows users to extract high-quality audio from video files using MoviePy.

## Features

🎬 Select any video file from your system

🎵 Extract high-quality audio

💾 Save output as MP3 file

🖥 Simple file selection interface (Tkinter)

⚡ Fast processing using MoviePy

## Technologies Used

Python 3.10+

MoviePy

Tkinter

NumPy

ImageIO / FFmpeg
How It Works

Tkinter opens a file dialog to select video

MoviePy loads the video file

Audio stream is extracted

Audio is saved as .mp3

## Requirements

Python 3.10.7

FFmpeg (installed automatically via imageio-ffmpeg dependency)

Internet not required

## Limitations

Output file name is currently fixed (demo.mp3)

No progress bar

No custom output folder selection
